# The Procurement Report Questionnaire

Use this checklist to understand how the client listens, what they watch, and where the system needs to go next. The goal is not more gear. The goal is better movies, better music, and fewer regret buys.

| Question / data point | Notes | Your notes / measurements |
| --- | --- | --- |
| Client name, location, contact number, and preferred WhatsApp number | Basic admin and follow-up. | |
| What are you trying to improve, replace, or build from scratch? | Start with the actual problem, not the shopping list. | |
| If this upgrade goes well, what should feel different on the first night? | Outcome-first opener. | |
| How will the system be used across movies, series, music, gaming, and sport? | Ask for rough percentages if useful. | |
| Which matters more right now: scale, clarity, bass impact, ease of use, or music quality? | Helps rank trade-offs. | |
| Is this for a dedicated room, lounge, media room, or open-plan space? | Room type shapes every recommendation. | |
| What are the room dimensions, or the closest estimate if exact figures are not available? | Enough to sense room volume and likely limits. | |
| How far do you sit from the screen or speakers now? | Useful for speaker sizing and display guidance. | |
| What equipment do you already own? | Record brand and model for every kept item. | |
| Which existing items must stay, and which are negotiable? | Protects sentimental or sunk-cost decisions without assuming all old gear should remain. | |
| Are there any current frustrations with the system? | Boomy bass, weak dialogue, too many remotes, HDMI glitches, harsh sound, not enough volume. | |
| Do you prefer a sound that feels crisp and forward, or warm and relaxed? | Keep the wording simple and human. | |
| Are you chasing cinema impact first, stereo music first, or a balanced all-rounder? | Useful for speaker and amplification priorities. | |
| Do you listen casually in the background, or do you sit down specifically to listen? | Helps judge user intent and system complexity. | |
| What music do you play most often? | Genre hints can shape tone and subwoofer expectations. | |
| Do you stream music, use local files, vinyl, TV apps, discs, or a mix? | Determines source and ecosystem needs. | |
| Do you use Apple, AirPlay, Sonos, Roon, HEOS, Spotify Connect, or another platform already? | Avoids recommending a system that fights the client’s habits. | |
| How comfortable are you with technology and setup menus? | Important when choosing simple versus fiddly system paths. | |
| Would you prefer one clean box where possible, or are you happy with separate components if the gains are worth it? | Good way to test appetite for complexity. | |
| Is there a display in place already, or is that part of the purchase decision too? | Establish whether the report includes picture hardware. | |
| For picture, what matters most: biggest possible image, deep blacks, daytime performance, or gaming features? | Clarifies projector versus TV direction. | |
| Are there windows or bright ambient light in the viewing space? | A practical filter for display choices. | |
| How many seats need a good experience, not just one sweet spot? | Influences speaker count and subwoofer strategy. | |
| Is future expansion likely in the next 12 to 24 months? | For example 3.1 to 5.1, or 5.1 to Atmos. | |
| Do you want the report to include phased upgrade options? | Helpful when budget is staged. | |
| What is the total budget range for hardware? | Encourage honesty so the report stays worth every rand. | |
| Is there a preferred split between audio, video, seating, or control? | Helps avoid overspending in the wrong place. | |
| Are you open to new, demo, or carefully selected used equipment? | Expands value options. | |
| Does the property have inverter or solar backup, or should power protection be part of the recommendation? | Important for local reliability. | |
| Are there brand preferences, or brands you absolutely want to avoid? | Note whether this is emotional, aesthetic, or based on past experience. | |
| Will you be buying soon, or are you planning ahead for later in the year? | Timing matters because stock and pricing move. | |
| Do you need South African stock and authorised warranty support only? | Filters out grey-market options. | |
| Are you working with an installer, handling this yourself, or asking a general electrician to help? | Changes how practical the recommendations need to be. | |
| Do you need advice on cables, rack location, or power as part of the shopping list? | Often missed until the end. | |
| Are there pets, children, or aesthetic constraints that affect speaker size or placement? | Real-world fit matters. | |
| Is there a hard limit in the room that rules certain options out? | Cabinet depth, wall space, wiring access, spouse approval, low ceilings, and so on. | |
| What would make this report feel genuinely useful to you? | A strong final alignment question. | |
