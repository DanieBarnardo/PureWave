# The Director Questionnaire

Use this checklist for full-scope projects where design decisions need to survive builders, timelines, and real-world compromise. The goal is not a pile of premium products. The goal is a room that works beautifully when the dust settles.

| Question / data point | Notes | Your notes / measurements |
| --- | --- | --- |
| Client name, project address, contact number, and preferred WhatsApp number | Confirm the principal contact first. | |
| Who are the decision-makers on this project? | Note homeowner, spouse, architect, builder, interior designer, and project manager. | |
| If this project goes exactly right, what should the room feel like when it is finished? | Start with the vision before the logistics. | |
| Is this a dedicated cinema, media room, or a larger entertainment space with multiple uses? | Shapes the overall design brief. | |
| What level of performance matters here: strong family-room performance, serious cinema, or reference-level ambition? | Helps position the scope without jargon. | |
| What stage is the build at today? | Concept, design development, shell, first-fix, fit-out, or rescue mission. | |
| What are the key programme dates from now to completion? | Include ceiling close-up, electrical first-fix, cabinetry, flooring, and commissioning dates. | |
| What is already signed off and hard to change? | Architecture, ceiling heights, window positions, joinery, HVAC routes, lighting layouts. | |
| Please list every stakeholder and their role | Also capture phone, email, and preferred communication style. | |
| Who approves technical changes on site when trade-offs appear? | Important for keeping momentum. | |
| Are there existing drawings available? | Plans, elevations, reflected ceiling plans, electrical, HVAC, joinery, and network layouts. | |
| Measure or confirm room length, width, height, and any stepped or irregular geometry | Physical reality before theory. | |
| What are the wall, floor, and ceiling constructions planned or already built? | Needed for isolation, acoustic treatment, and mounting strategy. | |
| What spaces sit next to, above, and below the room? | Bedrooms, studies, neighbours, roof spaces, garages, plant rooms. | |
| How important is sound isolation on this project? | Clarify whether the target is neighbour-friendly, family-friendly, or serious containment. | |
| What HVAC system is planned, and who is responsible for it? | Note diffuser positions, return-air strategy, and likely noise risk. | |
| Is there a target for low noise and quiet background performance? | Frame this in practical terms the team can understand. | |
| What lighting approach is planned in the room? | Downlights, cove lighting, step lights, wall washers, star ceilings, curtains, and scenes. | |
| Does the room need automation scenes? | Movie start, intermission, pause-for-doorbell, clean-up, security integration. | |
| Which control platform is planned, if any? | Control4, Savant, URC, app-only, or undecided. | |
| What network infrastructure is planned for the AV system? | Rack data points, display location, Wi-Fi coverage, switch location, and internet reliability. | |
| Is remote support or monitoring expected after handover? | Note whether a managed-support model is wanted. | |
| Where will the equipment rack live, and how will it be ventilated? | Capture access, noise, heat, and serviceability. | |
| What display strategy is under consideration? | Projector, direct-view LED, large TV, acoustically transparent screen, or undecided. | |
| How many seats are required, and which seats must feel best? | Include any future expansion. | |
| Is a riser, stage, bar counter, or custom seating layout planned? | Important for geometry and power planning. | |
| Are there aesthetic rules the room must respect? | Fabric walls, visible speakers, hidden tech, cabinet finishes, colour palette. | |
| What equipment is already owned, specified, or emotionally committed? | Record what is fixed versus flexible. | |
| What is the overall project budget, and what portion is allocated to AV, room build, seating, and automation? | Clarify hard limits early. | |
| Is there contingency built into the budget? | Essential on complex projects. | |
| Who pays third-party contractors, and who controls procurement? | Avoid confusion later. | |
| Are there preferred suppliers or contractors already appointed? | Useful for compatibility and communication. | |
| Does the electrician understand low-voltage separation, dedicated circuits, and clean cable routing? | A quiet room can be ruined by sloppy infrastructure. | |
| Are there power-quality concerns at the property? | Load shedding, inverter changeover, surge risk, generator noise, or grounding issues. | |
| Does the system need to integrate with solar, backup power, or generator logic? | Important in the local context. | |
| Are there security, access-control, or doorbell events that should interact with the room? | Helpful for automation logic. | |
| Are there environmental risks to note? | Damp, heat, sea air, dust, unstable power, construction delays. | |
| What are the top three project risks from the client’s point of view? | Good prompt for a risk register. | |
| What does success look like at handover? | Performance, usability, visual finish, reliability, and documentation. | |
| Who will look after the room after year one? | Sets up lifecycle and support expectations. | |
| What future upgrades should the room quietly prepare for now? | Extra channels, bigger screen, more subwoofers, new control, better network, and so on. | |
| What would make Pure Wave most valuable on this job? | End by aligning your oversight role with the real need. | |
