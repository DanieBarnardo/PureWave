# The Blueprint Questionnaire

Use this checklist during the site visit in the order that keeps the conversation clear and the room survey efficient. The goal is to leave with measured decisions instead of forum folklore.

| Question / data point | Notes | Your notes / measurements |
| --- | --- | --- |
| Client name, site address, contact number, and preferred WhatsApp number | Confirm the job record before starting. | |
| Who is making the final call on the room? | Useful when more than one household member is involved. | |
| If this room lands properly, what do you want to feel on night one? | Start with the client outcome, not the technical detail. | |
| What is the room mainly for: movies, mixed TV, gaming, music, or all of the above? | Shapes the balance between sound, screen, and usability. | |
| Is this a dedicated room or a shared living space? | Dedicated rooms allow more aggressive design moves. | |
| What is already fixed and cannot change? | Doors, windows, cabinetry, ceiling lines, furniture, and decor rules. | |
| What stage is the project at today? | Existing room, renovation, shell, first-fix, or nearly complete. | |
| What is the target timeline? | Important for sequencing wiring and contractor actions. | |
| Measure finished room length, width, and height | Take your own dimensions where possible. | |
| Note any alcoves, nibs, columns, beams, bulkheads, or ceiling drops | These often shape the whole layout. | |
| Are the walls parallel and symmetrical enough for a balanced layout? | Flag any asymmetry early. | |
| What are the wall constructions on each side of the room? | Brick, cavity wall, drywall, concrete, glazing, or mixed. | |
| What is the floor construction and finish? | Slab, timber, tile, carpet, laminate, rugs. | |
| What is the ceiling construction and available void depth? | Key for Atmos speakers, cable routes, and lighting. | |
| Where are all doors, windows, vents, and fixed joinery located? | Log positions carefully. | |
| How much light enters the room during daytime use? | Check natural light and practical blackout options. | |
| What colour are the ceiling, front wall, and side walls likely to be? | Light surfaces can hurt projection contrast. | |
| What is on the other side of each wall, above the ceiling, and below the floor? | Bedrooms, bathrooms, street side, roof, another lounge, and so on. | |
| Is isolation important here: keeping sound in, keeping noise out, or both? | Helps set the acoustic construction brief. | |
| Are there any rattles, vibration points, or noisy fittings already visible? | Note likely problem areas before recommending hardware. | |
| What HVAC or air movement is in the room now? | Split unit, ducted system, fans, vents, and their noise implications. | |
| Measure or note the ambient noise floor if possible | Even a rough reading adds value. | |
| Where does the client imagine the screen living? | Confirm preferred front wall and possible alternatives. | |
| What screen size does the client expect, and what seating distance feels realistic? | Match ambition to geometry. | |
| How many seats need to work well, and in what layout? | Single row, multiple rows, sectional couch, recliners, bar seating. | |
| Is a riser planned or possible? | Needed if multiple rows are under discussion. | |
| Where are the best candidate locations for the main listening position? | Mark options, not just one guess. | |
| Is there a preferred equipment rack location? | Record distance to front wall and service access. | |
| Are there current or planned conduits for HDMI, speaker cable, subwoofer cable, and network? | Verify rather than assume. | |
| Where are the current power outlets and circuits? | Map existing power before making demands of the electrician. | |
| Is a dedicated circuit realistic for the AV equipment and subwoofers? | Especially important for bigger systems. | |
| Are there existing network points in the room and at the proposed rack location? | Streaming and control reliability start here. | |
| Are any cables already installed that we need to keep, reuse, or replace? | Note type, route, and likely suitability. | |
| What existing gear is staying in the plan? | Record make, model, and physical placement constraints. | |
| What does the client dislike about the current setup or previous rooms? | Helps avoid repeating bad choices. | |
| What budget range is realistic for the next stage? | Useful for phased recommendations. | |
| Should the room be designed to grow later into more channels or more subwoofers? | Future-proofing question. | |
| Which contractors are involved already? | Builder, electrician, cabinetmaker, HVAC, automation, designer. | |
| Is there anything the contractors often get wrong on projects like this? | Good question when clients have prior build scars. | |
| What would a practical next step look like from the client’s point of view? | End by aligning the report to what helps them most. | |
