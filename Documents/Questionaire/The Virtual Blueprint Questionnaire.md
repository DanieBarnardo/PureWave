# The Virtual Blueprint Questionnaire

Use this checklist in sequence during the remote session. The goal is simple: get the room shape, priorities, and wiring decisions right before the build hardens.

| Question / data point | Notes | Your notes / measurements |
| --- | --- | --- |
| Client name, site address, contact number, and preferred WhatsApp number | Confirm who is driving decisions and where the room is located. | |
| Best email address for drawings, photos, and final report | Helps keep all supplied material in one place. | |
| If we get one thing right in this project, what matters most to you? | Start with the outcome they care about most. | |
| What does a great movie night look like in this room? | Listen for immersion, convenience, family use, aesthetics, or sheer scale. | |
| Is this room mainly for movies, mixed TV use, gaming, music, or all of the above? | Establish the content split early. | |
| How many people should the room serve comfortably most of the time? | Drives seating layout and speaker coverage. | |
| Is this a dedicated cinema room or a shared living space? | Shared spaces usually bring more compromises. | |
| What stage is the room at right now: concept, drawings, construction, renovation, or nearly finished? | This affects how much freedom remains. | |
| What is the deadline, if there is one? | Useful for recommending realistic wiring and upgrade phases. | |
| Please confirm the room length, width, and height to the nearest 10 mm | Ask for finished dimensions where possible, not rough shell sizes. | |
| Are any walls angled, curved, stepped, or not parallel? | Important for layout accuracy and room behavior. | |
| Are there bulkheads, beams, drop ceilings, pillars, or low points we need to work around? | These often shape speaker positions and screen options. | |
| Where are the doors, and which way do they swing? | Needed for seating flow, rack planning, and speaker clearance. | |
| Are there windows in the room, and which direction do they face? | Daylight control changes display strategy. | |
| Do you already have blackout curtains, blinds, or another light-control plan? | Helps judge projector viability. | |
| What are the walls made from? | Brick, block, drywall, concrete, or mixed construction all behave differently. | |
| What is the floor construction and floor finish? | Tile, timber, carpet, slab, and raised floors all change the room. | |
| What is the ceiling construction? | Ask whether it is slab, suspended, drywall, timber, or another system. | |
| Is there ceiling void space or any access above the ceiling? | Crucial for hidden cabling and in-ceiling speaker options. | |
| Has plastering, skimming, or closing-up already happened? | Tells you how much wiring freedom remains. | |
| Is there access below the floor or along skirtings for cable runs? | Gives fallback routing options if ceiling access is poor. | |
| Where is the display likely to go, and is the client leaning toward projector or TV? | Even if undecided, note the front-wall assumption. | |
| What screen size feels right to you right now? | Use this as a preference, then reality-check it later. | |
| How far from the screen do you imagine the main seats being? | Helps test viewing-angle expectations. | |
| Are you planning one row of seating or more than one? | If more than one, ask whether a riser is expected. | |
| Is there any furniture that must stay in the room? | Existing cabinetry and couches can lock in compromises. | |
| Where would you prefer the equipment to live: in-room, in a cabinet, in a rack, or elsewhere? | Needed for cable lengths, ventilation, and control logic. | |
| Where is the nearest practical power source for the display and equipment location? | Remote plans still need an infrastructure foundation. | |
| Are any speaker wires, HDMI runs, conduits, or network points already installed? | Existing infrastructure may help or create limits. | |
| If wiring is still open, are you willing to run extra cables now for future upgrades? | A key moment to protect future options. | |
| Would you like this room designed to grow later from a simpler system to a bigger one? | Capture upgrade logic early. | |
| Do you already own any equipment you definitely want to keep? | Record make, model, and condition where possible. | |
| What is the overall budget range for the room at this stage? | Keep it broad if the client is hesitant. | |
| Of that budget, what matters more right now: picture quality, sound quality, or infrastructure done properly from the start? | Helps with trade-offs when budget is tight. | |
| Does the property have inverter, solar backup, surge protection, or unstable mains power? | Important in the South African context. | |
| Are there any site access issues for large equipment delivery? | Staircases, tight corners, lifts, and parking matter more than clients expect. | |
| Who will use the report: the homeowner, builder, electrician, or another installer? | Helps pitch the final document at the right level. | |
| What photos can the client send today? | Ask for all four corners, front wall, rear wall, ceiling, floor, doors, windows, and current services. | |
| What drawings can the client send today? | Plans, elevations, reflected ceiling plans, and electrical layouts if available. | |
| Can the client record a quick ambient noise reading during a quiet period? | Even a phone reading is better than guessing. | |
| Is there anything about this room that already worries you? | Often surfaces the real constraint near the end. | |
| Confirm next step and missing information still needed | Close with a short action list. | |
